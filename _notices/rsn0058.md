## Overview

RAPIDS is dropping support for Python 3.10 in the upcoming release of v26.04, scheduled for April 8, 2026.  v26.04 will support Python 3.11, Python 3.13, and Python 3.13.


## Impact

Users should plan to move to a supported Python version 3.11, 3.12, or 3.13 as soon as possible.
