std::string cudf::io::parquet::ParquetMetadata::get_pandas_index_name ( )
inline

Extracts the column name used for the row indexes in a dataframe.

PANDAS adds its own metadata to the key_value section when writing out the dataframe to a file to aid in exact reconstruction. The JSON-formatted metadata contains the index column(s) and PANDA-specific datatypes.

Returns
std::string Name of the index column
