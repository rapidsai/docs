Detailed Description

This class is a collection of utilities for operating on words or tokens. It uses methods on the NVStrings class to access its character arrays.

◆ character_tokenize()

static NVStrings * NVText::character_tokenize ( NVStrings const & strings )
static

Tokenize all input strings into single-character strings.

Example

s = [ 'apple' , 'pear' ]
r = character_tokenize (s)
r is now [ 'a' , 'p' , 'p' , 'l' , 'e' , 'p' , 'e' , 'a' , 'r' ]

Note these are UTF-8 characters and so each string may be one or more bytes.

Parameters
strings Strings to tokenize
Returns
Strings with single characters.

◆ ngrams_tokenize()

static NVStrings * NVText::ngrams_tokenize ( NVStrings const & strs ,
const char * delimiter ,
int32_t ngrams ,
const char * separator
)
static

Creates ngrams of the tokens in each string.

Example:
s = [ 'a b' , 'c d e' ]
g = create_ngrams (s, ' ' ,2, '_' )
g is now ['a_b','c_d','d_e']
(note that there is no 'b_c' ngram in g)
Parameters
strs Strings to tokenize and make into ngrams.
delimiter String used to determine tokens. If null, whitespace is used as the delimiter.
ngrams The 'n' in ngrams. Example, use 2 for bigrams, 3 for trigrams, etc.
separator String used to join tokens.
Returns
The tokens as ngrams.
