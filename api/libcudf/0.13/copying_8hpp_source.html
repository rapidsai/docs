544 /*
545 * @brief Returns a new column, where each element is selected from either @p lhs or
546 * @p rhs based on the value of the corresponding element in @p boolean_mask
547 *
548 * Selects each element i in the output column from either @p rhs or @p lhs using the following rule:
549 * output[i] = (boolean_mask.valid(i) and boolean_mask[i]) ? lhs : rhs[i]
550 *
551 * @throws cudf::logic_error if lhs and rhs are not of the same type
552 * @throws cudf::logic_error if boolean mask is not of type bool8
553 * @throws cudf::logic_error if boolean mask is not of the same length as rhs
554 * @param[in] left-hand scalar
555 * @param[in] right-hand column_view
556 * @param[in] column of `BOOL8` representing "left (true) / right (false)" boolean for each element and
557 * null element represents false.
558 * @param[in] mr resource for allocating device memory
559 *
560 * @returns new column with the selected elements
561 */
