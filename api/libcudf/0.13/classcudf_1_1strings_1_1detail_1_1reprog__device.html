Detailed Description

Regex program of instructions/data for a specific regex pattern.

Once create, this find/extract methods are used to evaluating the regex instructions against a single string.

Create device program instance from a regex pattern.

The number of strings is needed to compute the state data size required when evaluating the regex.

Parameters
pattern The regex pattern to compile.
cp_flags The code-point lookup table for character types.
strings_count Number of strings that will be evaluated.

Does an extract evaluation using the compiled expression on the given string.

This will find a specific match within the string when more than match occurs.

Parameters
idx The string index used for mapping the state memory for this string in global memory (if necessary).
d_str The string to search.
[in,out] begin Position index to begin the search. If found, returns the position found in the string.
[in,out] end Position index to end the search. If found, returns the last position matching in the string.
column The specific instance to return if more than one match is found.
Returns
Returns 0 if no match is found.
