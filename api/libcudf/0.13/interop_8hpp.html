Detailed Description

Defines APIs for converting to/from "legacy" constructs such as gdf_column and gdf_dtype into their corresponding new constructs such as column_view and data_type .

------------------------------------------------------------------------—*

The purpose of these functions is to enable interoperability with APIs that

may not yet be updated to use the new constructs.
