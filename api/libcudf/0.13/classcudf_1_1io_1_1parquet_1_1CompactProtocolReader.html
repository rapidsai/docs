Detailed Description

Class for parsing Parquet's Thrift Compact Protocol encoded metadata.

This class takes in the starting location of the Parquet metadata, and fills out Thrift-derived structs and a schema tree.

In a Parquet, the metadata is separated from the data, both conceptually and physically. There may be multiple data files sharing a common metadata file.

The parser handles both V1 and V2 Parquet datasets, although not all compression codecs are supported yet.
