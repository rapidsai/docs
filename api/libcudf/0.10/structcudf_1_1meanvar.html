Detailed Description

template<typename ElementType>
struct cudf::meanvar< ElementType >

intermediate struct to calculate mean and variance This is an example case to output a struct from column input.

----------------------------------------------------------------------—* this will be used to calculate and hold sum of values, 'sum of squares', 'sum of valid count'. Those will be used to compute mean (= sum / count) and variance (= sum of squares / count - mean^2).

@tparam ElementType element data type of value and value_squared.
