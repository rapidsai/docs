Template specialization for operator() for types whose values can be convertible to a 0 or 1 to represent false/true. The converting is done by checking against the default and user-specified true/false values list.

------------------------------------------------------------------------—* It is handled here rather than within convertStrToValue() as that function

is used by other types (ex. timestamp) that aren't 'booleable'.
