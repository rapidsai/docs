Detailed Description

Helper struct that calculates the values needed to get quantile values by interpolation.

For a quantile that lies between indices i and j, this struct calculates i (lower_bound), j (upper_bound), index nearest to quantile between i and j (nearest), and the fractional distance that the quantile lies ahead i (fraction)
