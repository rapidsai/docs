template<typename T>
struct cudf::detail::unwrapped_type< T >

Trait to use to get underlying type of wrapped object.

------------------------------------------------------------------------—* This struct can be used with either a fundamental type or a wrapper type and it uses unwrap to get the underlying type.

Example use case: Making a functor to use with a type_dispatcher that works on the underlying type of all gdf_dtype

{c++}
struct example_functor{
template <typename T>
int operator()(){
using T1 = cudf::detail::unwrapped_type<T>::type;
return sizeof(T1);
}
};

@tparam T Either wrapped object type or fundamental type
