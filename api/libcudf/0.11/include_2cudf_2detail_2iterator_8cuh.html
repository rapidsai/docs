Detailed Description

provides column input iterator with nulls replaced with a specified value

-----------------------------------------------------------------------—*

The column input iterator is designed to be used as an input iterator for thrust and cub.

Usage: auto iter = make_null_replacement_iterator(column, null_value);

The column input iterator returns only a scalar value of data at [id] or the null_replacement value passed while creating the iterator. For non-null column, use auto iter = column.begin<Element>();

template<typename Element >
auto cudf::experimental::detail::make_null_replacement_iterator ( column_device_view const & column ,
Element const null_replacement = Element{0}
)

Constructs an iterator over a column's values that replaces null elements with a specified value.

Dereferencing the returned iterator for element i will return column[i] if it is valid, or null_replacement if it is null. This iterator is only allowed for nullable columns.

Exceptions
<tt>cudf::logic_error</tt> if the column is not nullable.
<tt>cudf::logic_error</tt> if column datatype and Element type mismatch.
