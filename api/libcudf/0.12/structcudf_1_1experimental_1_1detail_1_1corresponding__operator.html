template<aggregation::Kind k>
struct cudf::experimental::detail::corresponding_operator< k >

Maps an aggregation::Kind value to it's corresponding binary operator.

Note
Not all values of aggregation::Kind have a valid corresponding binary operator. For these values E, std::is_same_v<corresponding_operator<E>::type, void>.
