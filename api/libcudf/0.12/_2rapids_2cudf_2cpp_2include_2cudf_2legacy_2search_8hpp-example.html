Find if the value is present in the column and dtype of both value and column should match.------------------------------------------------------------------------—*

Exceptions
cudf::logic_errorIf dtype of column and value doesn't match

Single Column: idx 0 1 2 3 4 column = { 10, 20, 20, 30, 50 } Scalar: value = { 20 } result = true

Parameters
columnA gdf column
valueA scalar value to search for in column

@return bool If <tt>value</tt> is found in <tt>column</tt> true, else false.
