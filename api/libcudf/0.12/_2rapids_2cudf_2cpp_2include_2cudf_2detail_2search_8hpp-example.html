Find if the value is present in the col

Exceptions
cudf::logic_errorIf col.type() != values.type()

Single Column: idx 0 1 2 3 4 col = { 10, 20, 20, 30, 50 } Scalar: value = { 20 } result = true

Parameters
colA column object
valueA scalar value to search for in col
mrDevice memory resource to use for device memory allocation
streamStream to use for any kernel launches.
Returns
bool If value is found in column true, else false.
