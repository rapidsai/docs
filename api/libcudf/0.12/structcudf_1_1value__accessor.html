Detailed Description

template<typename ElementType, typename ResultType, bool has_nulls>
struct cudf::value_accessor< ElementType, ResultType, has_nulls >

value accessor with/without null bitmask A unary function returns scalar value at id. operator() (cudf::size_type id) computes data value and valid flag at id

----------------------------------------------------------------------—* If has_nulls = false, the data is always static_cast<ResultType>(data[id]) If has_nulls = true, the data value = (is_valid(id))? static_cast<ResultType>(data[id]) : identity;
