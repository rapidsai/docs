Detailed Description

template<typename Nullate, typename PhysicalEqualityComparator = nan_equal_physical_equality_comparator>
class cudf::experimental::row::equality::device_row_comparator< Nullate, PhysicalEqualityComparator >

Computes the equality comparison between 2 rows.

Equality is determined by comparing rows element by element. The first mismatching element returns false, representing unequal rows. If the rows are compared without mismatched elements, the rows are equal.

Template Parameters
NullateA cudf::nullate type describing whether to check for nulls.
PhysicalEqualityComparatorA equality comparator functor that compares individual values rather than logical elements, defaults to a comparator for which NaN == NaN.
