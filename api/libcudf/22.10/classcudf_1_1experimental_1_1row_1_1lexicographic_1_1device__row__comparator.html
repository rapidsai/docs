Detailed Description

template<bool has_nested_columns, typename Nullate, typename PhysicalElementComparator = sorting_physical_element_comparator>
class cudf::experimental::row::lexicographic::device_row_comparator< has_nested_columns, Nullate, PhysicalElementComparator >

Computes the lexicographic comparison between 2 rows.

Lexicographic ordering is determined by:

  • Two rows are compared element by element.
  • The first mismatching element defines which row is lexicographically less or greater than the other.
  • If the rows are compared without mismatched elements, the rows are equivalent

Lexicographic ordering is exactly equivalent to doing an alphabetical sort of two words, for example, aac would be less than (or precede) abb. The second letter in both words is the first non-equal letter, and a < b, thus aac < abb.

Template Parameters
