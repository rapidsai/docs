Detailed Description

template<typename T>
struct cudf::detail::value_accessor< T >

value accessor of column without null bitmask A unary functor returns scalar value at id. operator() (cudf::size_type id) computes element This functor is only allowed for non-nullable columns.

the return value for element i will return column[i]

Exceptions
cudf::logic_errorif the column is nullable.
cudf::logic_errorif column datatype and template T type mismatch.
