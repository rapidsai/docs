Detailed Description

template<typename T, gdf_dtype type_id>
struct cudf::detail::wrapper< T, type_id >

Base wrapper structure to emulate "strong typedefs" for gdf_dtype values that do not correspond to fundamental types.

Implements operators that allow the wrapper to be used as if it were a fundamental type.
