Detailed Description

overloads for CUDA atomic operations

------------------------------------------------------------------------* Provides the overloads for all of possible cudf's data types, where cudf's data types are, int8_t, int16_t, int32_t, int64_t, float, double, cudf::date32, cudf::date64, cudf::timestamp, cudf::category. where CUDA atomic operations are, atomicAdd , atomicMin , atomicMax , atomicCAS . Also provides cudf::genericAtomicOperation which performs atomic operation

with the given binary operator.

atomicAdd()

__forceinline__ __device__ int8_t atomicAdd ( int8_t * address ,
int8_t val
)

reads the old located at the address in global or shared memory, computes (old + val), and stores the result back to memory at the same address. These three operations are performed in one atomic transaction.

----------------------------------------------------------------------* The supported cudf types for atomicAdd are: int8_t, int16_t, int32_t, int64_t, float, double, cudf::date32, cudf::date64, cudf::timestamp, cudf::category. Cuda natively supports sint32 , uint32 , uint64 , float , `double, cudf::nvstring_category ( double is supported after Pascal). Other types are implemented by atomicCAS .
