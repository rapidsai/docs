Detailed Description

template<typename T, typename byte_t = unsigned char>
class gdf_table< T, byte_t >

The gdf_table class is meant to wrap a set of gdf_columns and provide functions for operating across all of the columns. It can be thought of as a matrix whose columns can be of different data types. Thinking of it as a matrix, many row-wise operations are defined, such as checking if two rows in a table are equivalent.
