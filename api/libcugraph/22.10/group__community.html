Compute Louvain.

Parameters
[in]handleHandle for accessing resources
[in]graphPointer to graph. NOTE: Graph might be modified if the storage needs to be transposed
[in]max_levelMaximum level in hierarchy
[in]resolutionResolution parameter (gamma) in modularity formula. This changes the size of the communities. Higher resolutions lead to more smaller communities, lower resolutions lead to fewer larger communities.
[in]do_expensive_checkA flag to run expensive checks for input arguments (if set to true)
[out]resultOutput from the Louvain call
[out]errorPointer to an error object storing details of any error. Will be populated if error code is not CUGRAPH_SUCCESS
Returns
