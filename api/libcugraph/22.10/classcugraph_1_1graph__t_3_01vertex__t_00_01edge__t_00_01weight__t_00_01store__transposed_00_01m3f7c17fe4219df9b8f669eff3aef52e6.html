Transpose the storage format (no change in actual graph).

In SG, convert between CSR and CSC. In multi-GPU, currently convert between CSR + DCSR hybrid and CSC + DCSC hybrid (but the internal representation in multi-GPU is subject to change).

Parameters
handleRAFT handle object to encapsulate resources (e.g. CUDA stream, communicator, and handles to various CUDA libraries) to run graph algorithms.
renumber_mapRenumber map to recover the original vertex IDs from the renumbered vertex IDs.
destroyIf true, destroy this graph to free-up memory.
Returns
std::tuple<graph_t<vertex_t, edge_t, weight_t, !store_transposed, multi_gpu>, rmm::device_uvector<vertex_t>> Return a storage transposed graph and a new renumber map (to recover the original vertex IDs for the returned graph).
