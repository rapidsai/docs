Labels each vertex in the input graph with its (strongly-connected-)component ID.

The input graph may be asymmetric. Component IDs can be arbitrary integers (they can be non-consecutive and are not ordered by component size or any other criterion).

Parameters
[in]handleHandle for accessing resources
[in]graphPointer to graph
[in]do_expensive_checkA flag to run expensive checks for input arguments (if set to true).
[out]resultOpaque pointer to labeling results
[out]errorPointer to an error object storing details of any error. Will be populated if error code is not CUGRAPH_SUCCESS
