Detailed Description

template<typename vertex_t, typename edge_t, typename mask_t = std::uint32_t>
struct cugraph::graph_mask_view_t< vertex_t, edge_t, mask_t >

Mask view to be used in device functions for reading and updating existing mask. This assumes the appropriate masks (vertex/edge) have already been initialized, since that will need to be done from the owning object.
