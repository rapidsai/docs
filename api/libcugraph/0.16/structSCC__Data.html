Detailed Description

template<typename ByteT, typename IndexT = int>
struct SCC_Data< ByteT, IndexT >

SCC Algorithm (Adapted from John Gilbert's "Graph Algorithms in the Language of Linear Algebra")

C = I | A | A^2 |...| A^k where matrix multiplication is through (combine, reduce) == (&, |) (bitwise ops) Then: X = C & transpose(C); apply get_labels(X);
