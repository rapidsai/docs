Detailed Description

Wrapper function for Nvgraph balanced cut clustering.

Exceptions
cugraph::logic_errorwhen an error occurs.
Template Parameters
VTType of vertex identifiers. Supported value : int (signed, 32-bit)
ETType of edge identifiers. Supported value : int (signed, 32-bit)
WTType of edge weights. Supported values : float or double.
Parameters
[in]graphinput graph object (CSR)
[in]num_clustersThe desired number of clusters
[in]num_eigen_vectsThe number of eigenvectors to use
[in]evs_toleranceThe tolerance to use for the eigenvalue solver
[in]evs_max_iterThe maximum number of iterations of the eigenvalue solver
[in]kmean_toleranceThe tolerance to use for the kmeans solver
[in]kmean_max_iterThe maximum number of iteration of the k-means solver
[out]clusteringPointer to device memory where the resulting clustering will be stored
