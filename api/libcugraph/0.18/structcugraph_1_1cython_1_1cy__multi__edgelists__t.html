Detailed Description

Owning struct. Allows returning multiple edge lists and edge offsets. cython only.

Parameters
number_of_verticesThe total number of vertices
number_of_edgesThe total number of edges (number of elements in src_indices, dst_indices and edge_data)
number_of_subgraphThe number of subgraphs, number of elements in subgraph_offsets - 1
source_indicesThis array of size E (number of edges) contains the index of the source for each edge. Indices must be in the range [0, V-1].
destination_indicesThis array of size E (number of edges) contains the index of the destination for each edge. Indices must be in the range [0, V-1].
edge_dataThis array size E (number of edges) contains the weight for each edge. This array can be null in which case the graph is considered unweighted.
subgraph_offsetsThis array size number_of_subgraph + 1 contains edge offsets for each subgraph
