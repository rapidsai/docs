Compute eigenvector centrality.

Computed using the power method.

Parameters

[in]graphPointer to graph
[in]epsilonError tolerance to check convergence. Convergence is measured comparing the L1 norm until it is less than epsilon

Compute hits.

Parameters
[in]handleHandle for accessing resources
[in]graphPointer to graph
[in]epsilonError tolerance to check convergence. Convergence is assumed if the sum of the differences in Hits values between two consecutive iterations is less than the number of vertices in the graph multiplied by epsilon.
[in]max_iterationsMaximum number of Hits iterations.
[in]initial_hubs_guess_verticesPointer to optional type erased device array containing the vertex ids for an initial hubs guess. If set to NULL there is no initial guess.
[in]initial_hubs_guess_valuesPointer to optional type erased device array containing the values for an initial hubs guess. If set to NULL there is no initial guess. Note that both initial_hubs_guess_vertices and initial_hubs_guess_values have to be specified (or they both have to be NULL). Otherwise this will be treated as an error.
[in]normalizeA flag to normalize the results (if set to true)
[in]do_expensive_checkA flag to run expensive checks for input arguments (if set to true).
[out]resultOpaque pointer to hits results
[out]errorPointer to an error object storing details of any error. Will be populated if error code is not CUGRAPH_SUCCESS
Returns

Compute katz centrality.

Parameters
[in]handleHandle for accessing resources
[in]graphPointer to graph
[in]betasOptionally send in a device array holding values to be added to each vertex's new Katz Centrality score in every iteration. If set to NULL then beta is used for all vertices.
[in]alphaKatz centrality attenuation factor. This should be smaller than the inverse of the maximum eigenvalue of this graph
[in]betaConstant value to be added to each vertex's new Katz Centrality score in every iteration. Relevant only when betas is NULL
[in]epsilonError tolerance to check convergence. Convergence is assumed if the sum of the differences in Katz Centrality values between two consecutive iterations is less than the number of vertices in the graph multiplied by epsilon. (L1-norm)
[in]max_iterationsMaximum number of Katz Centrality iterations.
[in]do_expensive_checkA flag to run expensive checks for input arguments (if set to true).
[out]resultOpaque pointer to katz centrality results
[out]errorPointer to an error object storing details of any error. Will be populated if error code is not CUGRAPH_SUCCESS

Parameters
