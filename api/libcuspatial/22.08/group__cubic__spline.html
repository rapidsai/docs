Create a table of cubic spline coefficients from columns of coordinates.

Computes coefficients for a natural cubic spline similar to the method found on http://mathworld.wolfram.com/CubicSpline.html .

The input data arrays t and y contain the vertices of many concatenated splines.

Currently, all input splines must be the same length. The minimum supported length is 5.

Note
Ids should be prefixed with a 0, even when only a single spline is fit, ids will be {0, 0}
Parameters
[in]tcolumn_view of independent coordinates for fitting splines
[in]ycolumn_view of dependent variables to be fit along t axis
[in]idsof incoming coordinate sets
[in]offsetsthe exclusive scan of the spline sizes, prefixed by 0. For example, for 3 splines of 5 vertices each, the offsets input array is {0, 5, 10, 15}.
[in]mrThe memory resource to use for allocating output
Returns
cudf::table_view of coefficients for spline interpolation. The size of the table is ((M-n), 4) where M is t.size() and and n is ids.size()-1.
