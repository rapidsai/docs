10 Minutes to cuDF ¶

Modeled after 10 Minutes to Pandas, this is a short introduction to cuDF, geared mainly for new users.

Object Creation ¶

Creating a Series . 

In [1]: s = cudf.Series([1,2,3,None,4])

In [2]: print(s)
      
0    1
1    2
2    3
3     
4    4

Creating a DataFrame by specifying values for each column. 

In [3]: df = cudf.DataFrame([('a', list(range(20))),
   ...: ('b', list(reversed(range(20)))),
   ...: ('c', list(range(20)))])
   ...: 

In [4]: print(df)

In [5]: pdf = pd.DataFrame({'a': [0, 1, 2, 3],'b': [0.1, 0.2, None, 0.3]})

Boolean Indexing ¶

Selecting rows in a Series by direct Boolean indexing. 

In [14]: print(df.b[df.b > 15])
      
0   19
1   18
2   17
3   16

Selecting values from a DataFrame where a Boolean condition is met, via the query API. 

In [15]: print(df.query("b == 3"))
     a    b    c
16   16    3   16

Supported logical operators include > , < , >= , <= , == , and != .

Join ¶

Performing SQL style merges. 

In [23]: df_a = cudf.DataFrame()

In [24]: df_a['key'] = [0, 1, 2, 3, 4]

In [25]: df_a['vals_a'] = [float(i + 10) for i in range(5)]

In [26]: df_b = cudf.DataFrame()

In [27]: df_b['key'] = [1, 2, 4]

In [28]: df_b['vals_b'] = [float(i+10) for i in range(3)]

In [29]: df_merged = df_a.merge(df_b, on=['key'], how='left')
1:float64
2:int64
3:float64

In [30]: print(df_merged.sort_values('key'))

3    0   10.0       
0    1   11.0   10.0
1    2   12.0   11.0
4    3   13.0       
2    4   14.0   12.0

Grouping ¶

Like pandas, cuDF supports the Split-Apply-Combine groupby paradigm. 

In [32]: df['agg_col1'] = [1 if x % 2 == 0 else 0 for x in range(len(df))]

In [33]: df['agg_col2'] = [1 if x % 3 == 0 else 0 for x in range(len(df))]

Grouping and then applying the sum function to the grouped data. 

In [34]: print(df.groupby('agg_col1').sum())
  agg_col1 sum_a sum_b sum_c sum_agg_col2
0        0   100    90   100            3
1        1    90   100    90            4

Grouping hierarchically then applying the sum function to grouped data. 

In [35]: print(df.groupby(['agg_col1', 'agg_col2']).sum())
  agg_col1 agg_col2 sum_a sum_b sum_c
0        0        0    73    60    73
1        0        1    27    30    27
2        1        0    54    60    54
3        1        1    36    40    36

Grouping and applying statistical functions to specific columns, using agg . 

In [36]: print(df.groupby('agg_col1').agg({'a':'max', 'b':'mean', 'c':'sum'}))
  agg_col1 max_a mean_b sum_c
0        0    19      9   100
1        1    18     10    90

Reshaping ¶

Time Series ¶

cuDF supports datetime typed columns, which allow users to interact with and filter data based on specific timestamps. 

In [37]: import datetime as dt

In [38]: date_df = cudf.DataFrame()

In [39]: date_df['date'] = pd.date_range('11/20/2018', periods=72, freq='D')

In [40]: date_df['value'] = np.random.sample(len(date_df))

In [41]: search_date = dt.datetime.strptime('2018-11-23', '%Y-%m-%d')

In [42]: print(date_df.query('date <= @search_date'))
                     date               value
0 2018-11-20T00:00:00.000 0.15416284237967237
1 2018-11-21T00:00:00.000  0.7400496965154048
2 2018-11-22T00:00:00.000 0.26331501518513467
3 2018-11-23T00:00:00.000  0.5337393933802977

Categoricals ¶

cuDF supports categorical columns. 

In [43]: pdf = pd.DataFrame({"id":[1,2,3,4,5,6], "grade":['a', 'b', 'b', 'a', 'a', 'e']})

In [44]: pdf["grade"] = pdf["grade"].astype("category")

In [45]: gdf = cudf.DataFrame.from_pandas(pdf)

In [46]: print(gdf)
  grade   id
0     a    1
1     b    2
2     b    3
3     a    4
4     a    5
5     e    6

Accessing the categories of a column. 

In [47]: print(gdf.grade.cat.categories)
('a', 'b', 'e')

Accessing the underlying code values of each categorical observation. 

In [48]: print(gdf.grade.cat.codes)
      
0    0
1    1
2    1
3    0
4    0
5    2
