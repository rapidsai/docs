Detailed Description

template<typename Upstream>
class rmm::mr::detail::arena::arena< Upstream >

An arena for allocating memory for a thread.

An arena is a per-thread or per-non-default-stream memory pool for handling small-size allocations. It allocates superblocks from the global arena, and return them when the superblocks become empty.

Template Parameters
UpstreamMemory resource to use for allocating the global arena. Implements rmm::mr::device_memory_resource interface.
