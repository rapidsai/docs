Detailed Description

An event logger for RMM.

----------------------------------------------------------------------—* Calling record() records various data about a memory manager event, includingthe type of event (alloc, free, realloc), the start and end time, the device, the pointer, free and total available memory, the size (for alloc/realloc), the stream, and location (file and line).

The log can be retreived as a CSV stream using to_csv().
