Find clusters with k-means algorithm. Initial centroids are chosen with k-means++ algorithm. Empty clusters are reinitialized by choosing new centroids with k-means++ algorithm.

Template Parameters

  • DataT – the type of data used for weights, distances.

  • IndexT – the type of data used for indexing.

Parameters

  • handle – [in] The raft handle.

  • params – [in] Parameters for KMeans model.

  • X – [in] Training instances to cluster. The data must be in row-major format. [dim = n_samples x n_features]

  • sample_weight – [in] Optional weights for each observation in X. [len = n_samples]

  • centroids – [inout] [in] When init is InitMethod::Array, use centroids as the initial cluster centers. [out] The generated centroids from the kmeans algorithm are stored at the address pointed by ‘centroids’. [dim = n_clusters x n_features]

  • inertia – [out] Sum of squared distances of samples to their closest cluster center.

  • n_iter – [out] Number of iterations run.

template<typename index_type_t, typename value_type_t> int kmeans (handle_t const &handle, index_type_t n, index_type_t d, index_type_t k, value_type_t tol, index_type_t maxiter, const value_type_t *__restrict__ obs, index_type_t *__restrict__ codes, value_type_t &residual, index_type_t &iters, unsigned long long seed=123456)

Find clusters with k-means algorithm. Initial centroids are chosen with k-means++ algorithm. Empty clusters are reinitialized by choosing new centroids with k-means++ algorithm.

Template Parameters

  • index_type_t – the type of data used for indexing.

  • value_type_t – the type of data used for weights, distances.

Parameters

  • handle – the raft handle.

  • n – Number of observation vectors.

  • d – Dimension of observation vectors.

  • k – Number of clusters.

  • tol – Tolerance for convergence. k-means stops when the change in residual divided by n is less than tol.

  • maxiter – Maximum number of k-means iterations.

  • obs – (Input, device memory, d*n entries) Observation matrix. Matrix is stored column-major and each column is an observation vector. Matrix dimensions are d x n.

  • codes – (Output, device memory, n entries) Cluster assignments.

  • residual – On exit, residual sum of squares (sum of squares of distances between observation vectors and centroids).

  • iters – on exit, number of k-means iterations.

  • seed – random seed to be used.

Returns

error flag
