Detailed Description

Build (i.e., fit, train) Decision Tree classifier for input data.

Parameters
[in]handleraft::handle_t
[in,out]treeCPU pointer to TreeMetaDataNode. User allocated.
[in]datatrain data (nrows samples, ncols features) in column major format, excluding labels. Device pointer.
[in]ncolsnumber of features (i.e., columns) excluding target feature.
[in]nrowsnumber of training data samples of the whole unsampled dataset.
[in]labels1D array of target features (int only). One label per training sample. Device pointer. Assumption: labels need to be preprocessed to map to ascending numbers from 0; needed for current gini impl. in decision tree.
[in,out]rowidsarray of n_sampled_rows integers in [0, nrows) range. Device pointer. The same array is then rearranged when splits are made, allowing us to construct trees without rearranging the actual dataset.
[in]n_sampled_rowsnumber of training samples, after sampling. If using decision tree directly over the whole dataset: n_sampled_rows = nrows
[in]n_unique_labelsnumber of unique label values. Number of categories of classification.
[in]tree_paramsDecision Tree training hyper parameter struct.
