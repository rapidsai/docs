Detailed Description

Parameters for kernel matrices. The following kernels are implemented:

  • LINEAR

    \[ K(x_1,x_2) = <x_1,x_2>, \]

    where \(< , >\) is the dot product
  • POLYNOMIAL

    \[ K(x_1, x_2) = (\gamma <x_1,x_2> + \mathrm{coef0})^\mathrm{degree} \]

  • RBF

    \[ K(x_1, x_2) = \exp(- \gamma |x_1-x_2|^2) \]

  • TANH

    \[ K(x_1, x_2) = \tanh(\gamma <x_1,x_2> + \mathrm{coef0}) \]
