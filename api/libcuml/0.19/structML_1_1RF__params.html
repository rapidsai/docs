◆ bootstrap

bool ML::RF_params::bootstrap

Control bootstrapping. If bootstrapping is set to true, bootstrapped samples are used for building each tree. Bootstrapped sampling is done by randomly drawing round(max_samples * n_samples) number of samples with replacement. More on bootstrapping: https://en.wikipedia.org/wiki/Bootstrap_aggregating If boostrapping is set to false, whole dataset is used to build each tree.
