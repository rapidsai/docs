Detailed Description

Parameters
cuml_handlereference to raft::handle_t object
Xdevice pointer to feature matrix of dimension NxD (row- or column major: see X_col_major param)
ydevice pointer to label vector of length N (for binary logistic: [0,1], for multinomial: [0,...,C-1])
Nnumber of examples
Dnumber of features
Cnumber of outputs (C > 1, for multinomial, indicating number of classes. For logistic and normal, C must be 1.)
fit_intercepttrue if model should include a bias. If true, the initial point/result w0 should point to a memory location of size (D+1) * C
l1l1 regularization strength (if non-zero, will run OWL-QN, else L-BFGS). Note, that as in scikit, the bias will not be regularized.
l2l2 regularization strength. Note, that as in scikit, the bias will not be regularized.
max_iterlimit on iteration number
grad_toltolerance for gradient norm convergence check
linesearch_max_itermax number of linesearch iterations per outer iteration
lbfgs_memoryrank of the lbfgs inverse-Hessian approximation. Method will request memory of size O(lbfgs_memory * D).
verbosityverbosity level
w0device pointer of size (D + (fit_intercept ? 1 : 0)) * C with initial point, overwritten by final result.
fhost pointer holding the final objective value
num_itershost pointer holding the actual number of iterations taken
X_col_majortrue if X is stored column-major, i.e. feature columns are contiguous
loss_typeid of likelihood model (0: logistic/sigmoid, 1: normal/squared, 2: multinomial/softmax)
