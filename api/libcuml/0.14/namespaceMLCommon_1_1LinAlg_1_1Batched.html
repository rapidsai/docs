Wrapper around cuBLAS batched gels (least-square solver of Ax=C)

: - This simple wrapper only supports non-transpose mode.

  • There isn't any strided version in cuBLAS yet.
  • cuBLAS only supports overdetermined systems.
  • This function copies A to avoid modifying the original one.
Parameters
[in] A Batched matrix A (must have more rows than columns)
[in,out] C Batched matrix C (the number of rows must match A)
